# 🎨 AI竞赛助手 - UI配色方案

## 配色理念

基于现代设计趋势和用户体验最佳实践，我们选择了**现代蓝色系**作为主配色方案，避免了传统的AI机器紫色，营造出专业、信任、科技感的视觉体验。

## 🎯 配色方案

### 主色调 - 现代蓝色系
- **主色**: `#2563eb` - 现代蓝，代表专业与科技
- **辅助色**: `#3b82f6` - 亮蓝，用于悬停和强调
- **深色**: `#1d4ed8` - 深蓝，用于文字和边框

### 功能色
- **成功色**: `#10b981` - 翠绿，表示成功状态
- **警告色**: `#f59e0b` - 琥珀，表示警告信息
- **危险色**: `#ef4444` - 红色，表示错误或危险
- **信息色**: `#06b6d4` - 青色，表示提示信息

### 中性色
- **主文字**: `#1e293b` - 深灰，用于标题和重要文字
- **次文字**: `#475569` - 中灰，用于正文
- **辅助文字**: `#64748b` - 浅灰，用于说明文字
- **禁用文字**: `#94a3b8` - 极浅灰，用于禁用状态

### 背景色
- **主背景**: `#ffffff` - 纯白，用于卡片和内容区域
- **次背景**: `#f8fafc` - 浅灰白，用于页面背景
- **三级背景**: `#f1f5f9` - 中灰白，用于悬停状态
- **四级背景**: `#e2e8f0` - 深灰白，用于边框和分割线

## 🌈 渐变效果

### 主要渐变
- **主渐变**: `linear-gradient(135deg, #2563eb 0%, #3b82f6 100%)`
- **成功渐变**: `linear-gradient(135deg, #10b981 0%, #34d399 100%)`
- **警告渐变**: `linear-gradient(135deg, #f59e0b 0%, #fbbf24 100%)`

### 应用场景
- 按钮背景
- 标题文字（使用 `background-clip: text`）
- 卡片悬停效果
- 进度条填充

## 📐 设计规范

### 圆角规范
- **小圆角**: `4px` - 用于小元素如标签
- **中圆角**: `8px` - 用于按钮和输入框
- **大圆角**: `12px` - 用于卡片
- **超大圆角**: `16px` - 用于对话框

### 阴影规范
- **小阴影**: `0 1px 2px 0 rgba(0, 0, 0, 0.05)`
- **中阴影**: `0 4px 6px -1px rgba(0, 0, 0, 0.1)`
- **大阴影**: `0 10px 15px -3px rgba(0, 0, 0, 0.1)`
- **超大阴影**: `0 20px 25px -5px rgba(0, 0, 0, 0.1)`

### 间距规范
- **超小间距**: `4px`
- **小间距**: `8px`
- **中间距**: `16px`
- **大间距**: `24px`
- **超大间距**: `32px`
- **特大间距**: `48px`

## 🎨 组件样式

### 按钮样式
- **主按钮**: 使用主渐变背景，悬停时上移2px
- **次按钮**: 使用浅色背景，悬停时变色
- **危险按钮**: 使用警告渐变背景

### 卡片样式
- **基础卡片**: 白色背景，浅色边框，小阴影
- **悬停效果**: 上移2px，增强阴影，边框变为主色
- **标题区域**: 使用渐变背景

### 输入框样式
- **基础状态**: 浅色边框
- **悬停状态**: 边框变为主色
- **聚焦状态**: 主色边框 + 主色阴影

### 表格样式
- **表头**: 浅色背景，深色文字
- **行悬停**: 浅色背景
- **边框**: 使用分割线颜色

## 📱 响应式设计

### 断点设置
- **移动端**: `< 768px`
- **平板端**: `768px - 1199px`
- **桌面端**: `≥ 1200px`

### 移动端适配
- 减少内边距
- 调整字体大小
- 优化按钮尺寸
- 简化布局结构

## 🎯 使用指南

### CSS变量使用
```css
/* 使用预定义的颜色变量 */
.my-element {
  color: var(--text-primary);
  background: var(--bg-primary);
  border: 1px solid var(--border-color);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-sm);
}
```

### 渐变文字效果
```css
.gradient-text {
  background: var(--gradient-primary);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}
```

### 悬停效果
```css
.hover-effect {
  transition: all 0.3s ease;
}

.hover-effect:hover {
  transform: translateY(-2px);
  box-shadow: var(--shadow-lg);
}
```

## 🔧 自定义主题

如需修改配色方案，只需更新 `colors.css` 文件中的 CSS 变量值即可：

```css
:root {
  --primary-color: #your-color;
  --primary-light: #your-light-color;
  /* ... 其他变量 */
}
```

## 📊 对比度检查

所有颜色组合都经过WCAG 2.1 AA级别对比度检查：
- 正常文字：对比度 ≥ 4.5:1
- 大文字：对比度 ≥ 3:1
- 非文字元素：对比度 ≥ 3:1
